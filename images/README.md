# 图片资源说明

本文件夹用于存放微信小程序所需的所有图片资源。请按照以下命名规则添加对应的图片文件：

## 所需图片列表

1. **底部导航栏图标**：
   - ai_teacher.png - AI老师未选中图标
   - ai_teacher_selected.png - AI老师选中图标
   - teacher_list.png - 教师名单未选中图标
   - teacher_list_selected.png - 教师名单选中图标
   - my.png - 我的未选中图标
   - my_selected.png - 我的选中图标

2. **头像图片**：
   - ai_avatar.png - AI老师头像
   - user_avatar.png - 用户头像
   - teacher1.png - 教师1头像
   - teacher2.png - 教师2头像
   - teacher3.png - 教师3头像

3. **功能图标**：
   - voice.png - 语音输入图标
   - arrow_right.png - 箭头图标
   - clear.png - 清除图标
   - about.png - 关于图标
   - feedback.png - 反馈图标
   - settings.png - 设置图标

## 图片规范

- 图标建议使用PNG格式，尺寸适中
- 头像建议使用正方形图片
- 为保证显示效果，请确保图片质量清晰
